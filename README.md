fortify-collector
=================

Fortify Collector for Hygieia


Sample Propertie:


dbname=dashboarddb
dbhost=localhost
dbusername=
dbpassword=
dbport=27017
logging.file=./logs/fortify-collector.log
fortify.cron=1 * * * * *
fortify.token=<token generated in Fortify>
fortify.server=http://myfortify.com/ssc
